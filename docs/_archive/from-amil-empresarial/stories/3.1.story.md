# Story 3.1: Template Dinâmico de Página Local + Dados de 500+ Cidades

> **Epic:** EPIC-001 — Site Amil Empresarial
> **Wave:** 3 — Expansão Local + Blog
> **Agente:** @dev + @data-engineer
> **Prioridade:** MUST | **Pontos:** 8
> **Status:** [ ] Planned

## Objetivo

Criar sistema de geração automática de páginas locais (estados + cidades) com dados reais de hospitais, Espaço Saúde e rede credenciada — a arma de dominação SEO local que nenhum concorrente possui.

## Acceptance Criteria

- [ ] Template gera página única por cidade com conteúdo diferenciado
- [ ] Dados reais de hospitais Amil por cidade (verificados)
- [ ] Schema LocalBusiness com endereço correto por cidade
- [ ] Build estático funcional para 500+ rotas
- [ ] Tempo de build < 5 minutos (ISR para cidades novas)
- [ ] 27 páginas de estado como hubs regionais
- [ ] 300+ páginas de cidade (100 grandes + 200 médias)
- [ ] Google Maps embed em cada página de cidade
- [ ] FAQ local com Schema em cada página
